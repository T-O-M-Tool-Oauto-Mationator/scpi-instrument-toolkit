# HP E3631A

Triple-output DC power supply (+6V/5A, +25V/1A, -25V/1A). Interface: GPIB.

```python
from lab_instruments import HP_E3631A
```

---

## Quick Example

```python
psu = HP_E3631A("GPIB0::5::INSTR")
psu.connect()

try:
    psu.set_output_channel(HP_E3631A.Channel.POSITIVE_6V, 3.3, current_limit=0.5)
    psu.enable_output(True)

    voltage = psu.measure_voltage(HP_E3631A.Channel.POSITIVE_6V)
    current = psu.measure_current(HP_E3631A.Channel.POSITIVE_6V)
    print(f"Output: {voltage:.3f} V, {current:.4f} A")
finally:
    psu.enable_output(False)
    psu.disconnect()
```

## Channels

| Channel Enum | SCPI Name | Voltage Range | Current Range |
|-------------|-----------|---------------|---------------|
| `HP_E3631A.Channel.POSITIVE_6V` | P6V | 0-6 V | 0-5 A |
| `HP_E3631A.Channel.POSITIVE_25V` | P25V | 0-25 V | 0-1 A |
| `HP_E3631A.Channel.NEGATIVE_25V` | N25V | 0-25 V | 0-1 A |

---

## Methods

### Output Control

| Method | Description |
|--------|-------------|
| `enable_output(enabled=True)` | Enable (`True`) or disable (`False`) the power supply output |
| `set_output_channel(channel, voltage, current_limit=None)` | Set voltage and current limit for a channel. Uses default current limit if `None`. |
| `set_voltage(channel, voltage)` | Set voltage only for the selected channel |
| `set_current_limit(channel, current)` | Set current limit only for the selected channel |
| `select_channel(channel)` | Select the active channel |
| `disable_all_channels()` | Disable output and zero all channel setpoints |

### Measurement

| Method | Returns | Description |
|--------|---------|-------------|
| `measure_voltage(channel)` | `float` | Measure output voltage (V) |
| `measure_current(channel)` | `float` | Measure output current (A) |

### Query

| Method | Returns | Description |
|--------|---------|-------------|
| `get_voltage_setpoint(channel=None)` | `float` | Query voltage setpoint |
| `get_current_limit(channel=None)` | `float` | Query current limit |
| `get_output_state()` | `bool` | Query whether output is enabled |
| `get_error()` | `str` | Read from the error queue |

### State Management

| Method | Description |
|--------|-------------|
| `set_tracking(enable)` | Enable/disable tracking mode for +/-25V supplies |
| `save_state(location)` | Save state to memory location (1, 2, or 3) |
| `recall_state(location)` | Recall saved state from memory location (1, 2, or 3) |

### Inherited from DeviceManager

| Method | Description |
|--------|-------------|
| `connect()` | Connect and initialize (clears status, disables all channels) |
| `disconnect()` | Close the VISA connection |
| `send_command(command)` | Send raw SCPI command |
| `query(command)` | Send SCPI query and return response |
| `reset()` | Reset instrument (`*RST`) |
| `clear_status()` | Clear status byte (`*CLS`) |

## Multi-Channel Example

```python
psu = HP_E3631A("GPIB0::5::INSTR")
psu.connect()

with psu:
    # Set all three channels
    psu.set_output_channel(HP_E3631A.Channel.POSITIVE_6V, 3.3, current_limit=1.0)
    psu.set_output_channel(HP_E3631A.Channel.POSITIVE_25V, 12.0, current_limit=0.5)
    psu.set_output_channel(HP_E3631A.Channel.NEGATIVE_25V, -5.0, current_limit=0.2)
    psu.enable_output(True)

    # Measure each channel
    for ch in [HP_E3631A.Channel.POSITIVE_6V, HP_E3631A.Channel.POSITIVE_25V, HP_E3631A.Channel.NEGATIVE_25V]:
        v = psu.measure_voltage(ch)
        i = psu.measure_current(ch)
        print(f"{ch}: {v:.3f} V, {i:.4f} A")
```
