# JDS6600

Dual-channel DDS function generator (up to 60 MHz). Interface: Serial (115200 baud).

```python
from lab_instruments import JDS6600_Generator
```

!!! note "Non-SCPI Protocol"
    The JDS6600 uses a proprietary ASCII protocol over USB serial, not standard SCPI. The driver handles all protocol details.

---

## Quick Example

```python
awg = JDS6600_Generator("ASRL4::INSTR")
awg.connect()

try:
    awg.set_waveform(1, "sine")
    awg.set_frequency(1, 1000)      # 1 kHz
    awg.set_amplitude(1, 2.0)       # 2 Vpp
    awg.enable_output(ch1=True)
finally:
    awg.disable_output()
    awg.disconnect()
```

## Waveforms

17 built-in waveforms:

`sine`, `square`, `pulse`, `triangle`, `partial_sine`, `cmos`, `dc`, `half_wave`, `full_wave`, `pos_ladder`, `neg_ladder`, `noise`, `exp_rise`, `exp_decay`, `multitone`, `sinc`, `lorenz`

## Methods

### Output Control

| Method | Description |
|--------|-------------|
| `enable_output(ch1=None, ch2=None)` | Enable/disable channels. `None` keeps current state. |
| `disable_output()` | Zero all parameters and disable both outputs |

### Waveform Configuration

| Method | Description |
|--------|-------------|
| `set_waveform(channel, waveform)` | Set waveform type (see list above) |
| `set_frequency(channel, freq_hz)` | Set frequency in Hz (0.01 Hz to 60 MHz) |
| `set_amplitude(channel, amplitude_v)` | Set amplitude in Vpp (0.001 to ~20V) |
| `set_offset(channel, offset_v)` | Set DC offset (-10V to +10V) |
| `set_duty_cycle(channel, duty_percent)` | Duty cycle for pulse/square (0.1-99.9%) |
| `set_phase(channel, phase_deg)` | Set phase (0-359.9 degrees) |

### Channel Synchronization

| Method | Description |
|--------|-------------|
| `set_sync(freq=False, waveform=False, amplitude=False, offset=False, duty=False)` | Sync CH2 to follow CH1 for selected parameters |

### Query (Cached Values)

| Method | Returns | Description |
|--------|---------|-------------|
| `get_amplitude(channel)` | `float` or `None` | Cached amplitude |
| `get_offset(channel)` | `float` or `None` | Cached offset |
| `get_frequency(channel)` | `float` or `None` | Cached frequency |
| `get_output_state(channel)` | `bool` | Cached output state |

## Dual-Channel Example

```python
awg = JDS6600_Generator("ASRL4::INSTR")
awg.connect()

try:
    # CH1: 1 kHz sine, 2 Vpp
    awg.set_waveform(1, "sine")
    awg.set_frequency(1, 1000)
    awg.set_amplitude(1, 2.0)

    # CH2: 1 kHz square, 3.3 Vpp
    awg.set_waveform(2, "square")
    awg.set_frequency(2, 1000)
    awg.set_amplitude(2, 3.3)

    # Enable both
    awg.enable_output(ch1=True, ch2=True)
finally:
    awg.disable_output()
    awg.disconnect()
```

---

## Channel Sync Example

```python
# Make CH2 mirror CH1's frequency and waveform
awg.set_sync(freq=True, waveform=True)
```

---

## Notes

- Serial connection at 115200 baud, 8N1.
- Query methods return cached values from the last `set_*` call, not live queries from the device.
- Frequency resolution is 0.01 Hz for frequencies below 20 MHz.
- The `enable_output()` method controls both channels simultaneously.
