# BK Precision 4063

Dual-channel arbitrary waveform generator. Interface: USB.

```python
from lab_instruments import BK_4063
```

---

## Quick Example

```python
awg = BK_4063("USB0::0xF4EC::0x1101::SPD12345678::INSTR")
awg.connect()

try:
    awg.set_waveform(1, "SINE", frequency=1000, amplitude=2.0, offset=0)
    awg.enable_output(1, True)
finally:
    awg.disable_all_channels()
    awg.disconnect()
```

---

## Context Manager

```python
awg = BK_4063("USB0::0xF4EC::0x1101::SPD12345678::INSTR")
awg.connect()

with awg:
    awg.set_waveform(1, "SQUARE", frequency=5000, amplitude=3.3, duty=50)
    awg.enable_output(1, True)
# Outputs disabled on exit
```

---

## Valid Waveforms

`SINE`, `SQUARE`, `RAMP`, `PULSE`, `NOISE`, `DC`, `ARB`

Aliases accepted: `SIN`, `SQU`, `PULS`, `NOIS`, `TRI`/`TRIANGLE` (mapped to RAMP).

---

## Methods

### Output Control

| Method | Description |
|--------|-------------|
| `enable_output(channel, enabled=True)` | Enable/disable output for channel 1 or 2 |
| `disable_all_channels()` | Zero and disable both outputs |
| `set_output_impedance(channel, load)` | Set load impedance (ohms or `"HZ"` for high-Z) |
| `set_sync_output(channel, enabled)` | Enable/disable sync output |

### Waveform Configuration

| Method | Description |
|--------|-------------|
| `set_waveform(channel, wave_type, frequency=None, amplitude=None, offset=None, phase=None, duty=None, symmetry=None)` | Set waveform type and all parameters |
| `set_dc_output(channel, voltage)` | Set DC output |

### Modulation, Sweep & Burst

| Method | Description |
|--------|-------------|
| `set_modulation(channel, state, mod_type="AM", source="INT", **kwargs)` | Configure modulation (AM, FM, PM, FSK, ASK, DSBAM, PWM) |
| `set_sweep(channel, state, **kwargs)` | Frequency sweep (TIME, START, STOP, etc.) |
| `set_burst(channel, state, **kwargs)` | Burst mode (MODE, PRD, STPS, etc.) |

### Query

| Method | Returns | Description |
|--------|---------|-------------|
| `get_amplitude(channel)` | `float` | Current amplitude (Vpp) |
| `get_offset(channel)` | `float` | Current DC offset |
| `get_frequency(channel)` | `float` | Current frequency (Hz) |
| `get_output_state(channel)` | `bool` | Whether output is enabled |
| `get_error()` | `str` | Error queue |

### Utility

| Method | Description |
|--------|-------------|
| `copy_channel(dest_channel, src_channel)` | Copy parameters between channels |

---

## Notes

- Uses BK Precision 4060 series command syntax (`C1:BSWV WVTP,SINE,FRQ,1000,...`).
- The `set_modulation`, `set_sweep`, and `set_burst` methods accept keyword arguments that map directly to the instrument's parameter names.
