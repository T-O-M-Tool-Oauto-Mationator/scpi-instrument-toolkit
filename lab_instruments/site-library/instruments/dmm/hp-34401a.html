    HP 34401A¶

    6.5-digit digital multimeter. Interface: GPIB.

    from lab_instruments import HP_34401A

    Quick Example¶

    dmm = HP_34401A("GPIB0::22::INSTR")
dmm.connect()

try:
    dmm.configure_dc_voltage()
    voltage = dmm.read()
    print(f"DC Voltage: {voltage:.6f} V")
finally:
    dmm.disconnect()

    Context Manager¶

    dmm = HP_34401A("GPIB0::22::INSTR")
dmm.connect()

with dmm:
    dmm.configure_dc_voltage(range_val=10, nplc=10)
    reading = dmm.read()
    print(f"{reading:.6f} V")
# Instrument reset on exit

    Two Ways to Measure¶

    Configure + Read (recommended for repeated measurements):

    dmm.configure_dc_voltage()    # Configure once
reading1 = dmm.read()         # Trigger and read
reading2 = dmm.read()         # Read again (same config)

    Immediate Measure (one-shot, reconfigures each time):

    voltage = dmm.measure_dc_voltage()   # Configure + trigger + return

    Methods¶

    Configuration¶

    Method Description
    configure_dc_voltage(range_val="DEF", resolution="DEF", nplc=None) Configure for DC voltage
    configure_ac_voltage(range_val="DEF", resolution="DEF") Configure for AC voltage
    configure_dc_current(range_val="DEF", resolution="DEF", nplc=None) Configure for DC current
    configure_ac_current(range_val="DEF", resolution="DEF") Configure for AC current
    configure_resistance_2wire(range_val="DEF", resolution="DEF", nplc=None) Configure for 2-wire resistance
    configure_resistance_4wire(range_val="DEF", resolution="DEF", nplc=None) Configure for 4-wire resistance
    configure_frequency(range_val="DEF", resolution="DEF") Configure for frequency
    configure_period(range_val="DEF", resolution="DEF") Configure for period
    configure_continuity() Configure for continuity test
    configure_diode() Configure for diode test

    NPLC (Number of Power Line Cycles) controls integration time. Valid values: 0.02, 0.2, 1, 10, 100. Higher = less noise, slower.

    Reading¶

    Method Returns Description
    read() float Trigger a measurement and return the result
    fetch() float Return the last measurement without triggering

    Immediate Measurement¶

    Method Returns Description
    measure_dc_voltage(range_val, resolution) float One-shot DC voltage
    measure_ac_voltage(range_val, resolution) float One-shot AC voltage
    measure_dc_current(range_val, resolution) float One-shot DC current
    measure_ac_current(range_val, resolution) float One-shot AC current
    measure_resistance_2wire(range_val, resolution) float One-shot 2-wire resistance
    measure_resistance_4wire(range_val, resolution) float One-shot 4-wire resistance
    measure_frequency(range_val, resolution) float One-shot frequency
    measure_period(range_val, resolution) float One-shot period
    measure_continuity() float One-shot continuity (ohms)
    measure_diode() float One-shot diode voltage

    Trigger¶

    Method Description
    set_trigger_source(source="IMM") Set trigger source: IMM, BUS, or EXT
    set_trigger_delay(delay="MIN") Set trigger delay (0-3600 s)
    set_trigger_count(count=1) Set number of triggers to accept
    set_sample_count(count=1) Set samples per trigger (1-50000)
    trigger() Send software trigger (*TRG)
    init() Arm the trigger (idle to wait-for-trigger)

    Display & Utility¶

    Method Description
    set_display(enabled) Enable/disable front panel display
    display_text(text) Show custom text (max 12 chars)
    display_text_rolling(text, width, delay, pad, loops) Scroll text across display
    clear_display_text() Clear custom text
    set_beeper(enabled) Enable/disable beeper
    beep() Sound the beeper once
    get_error() Read from error queue

    NPLC Example¶

    dmm = HP_34401A("GPIB0::22::INSTR")
dmm.connect()

with dmm:
    # Fast measurement (20ms integration)
    dmm.configure_dc_voltage(nplc=0.02)
    fast = dmm.read()

    # High-accuracy measurement (2s integration at 50Hz mains)
    dmm.configure_dc_voltage(nplc=100)
    accurate = dmm.read()

    print(f"Fast: {fast:.6f} V")
    print(f"Accurate: {accurate:.6f} V")

    Multi-Sample Example¶

    dmm = HP_34401A("GPIB0::22::INSTR")
dmm.connect()

with dmm:
    dmm.configure_dc_voltage()
    dmm.set_sample_count(10)         # 10 samples per trigger
    reading = dmm.read()             # Returns average of 10 samples
    print(f"Average: {reading:.6f} V")
